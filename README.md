## ✨ 功能特性

- 📺 **自动播放**：进入课程后自动静音播放，无需人工干预。
- ⏭️ **自动连播**：当前视频播放完毕后，自动跳转至下一节任务。
- 🔧 **智能时长**：自动读取视频时长或手动修正，防止进度未记录。
- 🧩 **悬浮面板**：右侧内置悬浮球，支持双击展开/折叠控制面板。

## 📖 使用方法

1. 首先安装浏览器扩展 **[ScriptCat (脚本猫)](https://docs.scriptcat.org/)**。
2. 点击上方的 **[安装脚本]** 按钮。
3. 在打开的页面中点击 **“安装脚本”**。
4. 打开小雅课程页面，脚本将自动运行（右侧会出现悬浮球）。
